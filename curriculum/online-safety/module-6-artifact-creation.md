# Module 6: Digital Artifact Creation

**Objective:** Learn to create, secure, and manage digital assets as part of your professional digital legacy.

## Key Concepts
- **Artifact Ownership:** Understanding IP (Intellectual Property) and how to prove you created your work.
- **Secure Creation:** Using tools that don't leak metadata or private information during the creative process.
- **Sustainable Assets:** Choosing formats and platforms that will last (e.g., decentralized storage, GitHub repositories).

## Technical Skills
- Setting up a secure GitHub repository for student projects.
- Understanding digital signatures and basic metadata stripping from creative files.
- Introduction to decentralized storage (IPFS) for long-term artifact preservation.
